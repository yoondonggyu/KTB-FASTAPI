## 1. 개요

### 1.1 프로젝트 구조
```
FASTAPI_PROJECT_KTB/
├── FASTAPI_Project_back/    # 백엔드 API 서버
├── FASTAPI_Project_model/   # AI 모델 서빙 서버
└── FASTAPI_Project_front/   # 프론트엔드
```

### 1.2 테스트 목적
- API 엔드포인트 정상 동작 확인
- 입력 유효성 검사 검증
- 에러 처리 확인
- 인증/인가 동작 검증
- AI 모델 서빙 기능 확인
