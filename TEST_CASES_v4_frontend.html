테스트 특징

프론트엔드 테스트는 JavaScript 로직을 Python으로 동일하게 검증합니다.

  • 유효성 검사: 이메일, 비밀번호, 닉네임 형식 검증
  • 응답 구조: API 응답 형식 및 데이터 구조 검증
  • 비즈니스 로직: 좋아요, 조회수, 권한 등 로직 검증
  • 에러 처리: 에러 메시지 및 응답 구조 검증

테스트 환경

  • 프레임워크: pytest
  • 경로: FASTAPI_Project_front/tests/
  • 실행 명령: pytest FASTAPI_Project_front/tests/ -v
